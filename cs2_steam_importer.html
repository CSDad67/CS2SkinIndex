Steam Inventory Importer

// IMPORT YOUR CS2 COLLECTION IN 3 STEPS — NO API KEY REQUIRED //

HOW IT WORKS
The bookmarklet below runs directly on your Steam inventory page while you are logged in.
It uses your existing Steam session — no API key, no passwords, nothing is stored or transmitted.
The data never leaves your browser. It generates a JSON file you paste into the Inventory Manager.
1
Drag the button below to your browser's bookmarks bar
(If you don't see the bookmarks bar: press Ctrl+Shift+B in Chrome/Firefox to show it)
◆ CS2 Skin Index — Import Inventory
⇧ Drag this button to your bookmarks bar  |  or right-click → Bookmark this link
2
Go to your Steam inventory page and click the bookmarklet
Open: steamcommunity.com/my/inventory  (log in if needed)
Make sure CS2 is selected (the default tab). Then click CS2 Skin Index — Import Inventory in your bookmarks bar.
A panel will appear at the bottom of the page showing your skins being processed.
3
Copy the generated JSON and paste it into the Inventory Manager
The bookmarklet shows a text box with your inventory JSON.
Click the box → Ctrl+A → Ctrl+C to copy.
Then open cs2_inventory_manager.html → paste into the Import box → click MERGE IMPORT.
ALTERNATIVE — PASTE STEAM INVENTORY JSON DIRECTLY
If the bookmarklet doesn't work, you can get your inventory JSON manually:
1. While logged into Steam, open this URL in your browser (replace YOUR_STEAMID):
https://steamcommunity.com/inventory/YOUR_STEAMID/730/2?l=english&count=5000
2. You'll see raw JSON. Press Ctrl+A then Ctrl+C to copy it.
3. Paste below and click PROCESS & IMPORT.
Ready to import! Click inside the box → Ctrl+A → Ctrl+C → open Inventory Manager → paste → MERGE IMPORT
△ Click box → Ctrl+A → Ctrl+C  |  Mac: Cmd+A, Cmd+C
